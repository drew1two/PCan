PCan
====

A community climate action website, using the Phalcon framework, started out from Vokura
Development site is hosted at parracan.net.

To get it going
* Create a mysql database, with unique user with rights and a password.
* Run sql script pcan.sql on new database to create tables and population data.
* Copy sample_config.php to config.php
* Populate database connection data and other particulars in config.php
* Set up configuration for a webserver. Developed and tested on nginx.
